// Installation

Prêt en 5 minutes

1

Installer HACS

Si HACS n'est pas encore installé, rendez-vous sur hacs.xyz et suivez le guide officiel. HACS est le gestionnaire de composants communautaires pour Home Assistant.

2

Ajouter le dépôt personnalisé

HACS → Intégrations → ⋮ → Dépôts personnalisés
URL : https://github.com/JeanMarc-Labs/ha-config-auditor
Catégorie : Intégration

3

Installer H.A.C.A

Recherchez H.A.C.A dans la liste HACS, cliquez Télécharger, puis redémarrez Home Assistant.

4

Configurer l'intégration

Paramètres → Appareils & Services → + Ajouter → H.A.C.A
Suivez le wizard. Le panneau H.A.C.A apparaît dans votre sidebar. ✅

📁
Installation manuelle : Copiez le dossier custom_components/config_auditor/ dans /config/custom_components/ et redémarrez HA.

// Features

Everything you need
in a single panel

H.A.C.A continuously analyzes your installation and acts where needed — quietly in the background.

🔍

Continuous Audit

Automatic scan every 60 min. Real-time health score. No action required.

⚡

1-Click Fixes

Before/after preview. Direct application. Automatic YAML backup before any change.

🤖

Built-in AI

Issue explanations, description suggestions, complex automation optimization with your HA LLM.

📊

History & Trends

Track score evolution over time. Sparkline chart. Export reports as PDF, Markdown or JSON.

🔋

Battery Monitor

Monitor all battery entities. Configurable alerts. Sensors exposed in HA.

🗄️

Recorder DB

Detect orphan entities in the database. Safe selective purge.

🛡️

Security

Hardcoded secrets, exposed sensitive data, and vulnerable configurations detection.

🗺️

Dependency Map

Visual graph of links between entities, automations and scripts.

✅

Compliance

Metadata audit: friendly names, missing areas, unused labels, helpers without description. AI-powered fixes included.

🧩

Helpers

Dedicated tab for all your input helpers (input_boolean, timer, counter…) with detection of helpers never used in automations.

💬

AI Chat

Conversational assistant aware of your health score, critical issues and HA configuration. Available from the AI panel.

🔗

MCP Server

Built-in Model Context Protocol server exposing H.A.C.A tools to external AI agents (Claude Desktop, Cursor…).

📜

Script Graph

Call chain analysis: cycle detection, depth > 3 levels, orphaned scripts, blueprint candidate detection.

👥

Group Analyzer

Detects empty groups, missing references, all-unavailable member entities, nested groups exceeding 2 levels.

🗺️

Area Complexity v1.5

Interactive complexity heatmap per area. Composite score. Automatic merge and split suggestions via AI Chat.

🔄

Redundancy Analyzer v1.5

Detects logical overlaps, blueprint candidates (≥3 identical automations) and automations replaceable by native HA features.

🔮

Battery Predictions v1.5

Linear regression on HA history. Estimated replacement dates. 7-day advance alerts. Full CSV export.

📉

Recorder Impact v1.5

Ranked by writes/day and MB/year. Generates a copy-paste-ready recorder: exclude: YAML block.

🌍

13 Languages

Interface available in FR, EN, DE, ES, IT, NL, PT, PL, DA, SV, RU, JA and ZH. Use haca_ignore label to exclude entities.

// Installation

Ready in 5 minutes

1

Install HACS

If HACS isn't installed yet, visit hacs.xyz and follow the official guide. HACS is the community store for Home Assistant components.

2

Add the custom repository

HACS → Integrations → ⋮ → Custom repositories
URL: https://github.com/JeanMarc-Labs/ha-config-auditor
Category: Integration

3

Install H.A.C.A

Find H.A.C.A in the HACS list, click Download, then restart Home Assistant.

4

Configure the integration

Settings → Devices & Services → + Add Integration → H.A.C.A

📁
Manual install: Copy the custom_components/config_auditor/ folder to /config/custom_components/ and restart HA.
